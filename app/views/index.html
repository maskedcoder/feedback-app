Simple Angular Example App

This example Angular app was hacked together in a few short coding sessions over the course of a few days and is likely still not done.

In the final version of this app, the user should be able to:

  1. View a list of products/businesses
  2. Comment on said products/businesses
  3. Mark comments as helpful/unhelpful

Enough talk. Let's see it go.
